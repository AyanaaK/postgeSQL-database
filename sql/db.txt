База данных - отдельная программа предназначенная для постоянного хранения
информации и предоставляющая инструменты для взаимодействия с ней и со
структурами в которых эта информация хранится. Большинство баз данных работают
по принципу клиент-сервер, где сервер выступает в качестве непосредственного
хранилища информации и обрабатывает входящие от клиентов запросы на
взаимодействие, клиент в свою очередь позволяет построить подключение к серверу
и сформировать запросы для их выполнения на сервере. В качестве клиента для
базы данных PostgreSQL могут использоваться следующие программы: psql, pgAdmin,
IDEA и так далее.

Реляционные базы данных - базы данных в которых конечной точкой хранения
информации выступает структура под названием таблица, таблица в свою очередь
состоит из полей (столбцы) и записей (ряды), каждое поле таблицы имеет свой
строго определённый при создании тип данных. В реляционных базах данных важной
особенностью является построение связей между таблицами (ситуация при которой
запись одной таблицы может ссылаться на запись другой таблицы). Реляционных баз
данных большое множество (MySQL, PostgreSQL, Oracle Database и так далее), но
для взаимодействия с ними применяется один и тот же язык под названием SQL. SQL
как язык стандартизирован, но может незначительно отличится для каждой отдельно
взятой базы данных, такой вариант языка называется SQL диалектом базы данных
(например язык SQL для базы данных PostgreSQL корректнее называть PostgreSQL
диалект).

* Клиент:
- Открытие сессии в контексте которой будут выполниться запросы к серверу.
- Написание SQL запроса и отправка его на выполнение в контексте открытой
сессии.

Сервер:
- Принимает входящий SQL запрос, обрабатывает его и отправляет ответ клиенту с
результатом выполнения.

Клиент:
- Принимает ответ сервера на отправленный SQL запрос и отображает его в
программе клиенте.

Данные необходимые для подключения к серверу базы данных PostgreSQL:
1) Хост - адрес компьютера (ip либо доменное имя) на котором работает сервер
базы данных для подключения. Если сервер базы данных находится на рабочем
компьютере разработчика необходимо указать ip 127.0.0.1 либо доменное имя
localhost, это адреса ссылающиеся на текущее устройство.
2) Порт - номер виртуального сетевого интерфейса от 0 до 65535 к которому
привязан требуемый для подключения сервер базы данных. По умолчанию база данных
PostgreSQL занимает порт 5432, но порт можно задать на своё усмотрение в файле
конфигурации либо при установке.
3) Имя пользователя - имя пользователя от лица которого необходимо построить
подключение. В PostgreSQL по умолчанию создаётся пользователь с неограниченным
набором прав, называется такой пользователь postgres.
4) Пароль пользователя - пароль пользователя от лица которого необходимо
построить подключение.
